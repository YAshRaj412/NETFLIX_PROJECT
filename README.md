
## Netflix Content & Viewership Analysis

### Objective
Analyze Netflix content strategy, engagement patterns, and profitability using SQL.

### Datasets
- Content metadata
- Viewership data
- Production & marketing cost data

### Key Analysis
- Movies vs TV Shows
- Regional content dominance
- Genre trends
- Profitability ranking
- Viewer engagement
- Risk categorization

### Tools Used
- SQL (CTEs, Joins, Window Functions)
- Data Visualization
- Business Analytics

### Key Insights
- TV Shows drive retention
- Few titles generate majority of profit
- Cost efficiency > high budget
- India is a key growth market
